# array-random
> Returns a random item from an array.

## Install

Install with [npm](https://www.npmjs.com/):
```sh
$ npm install --save array-random
```

## Usage

```js
const arrayRandom = require('array-random);

console.log(arrayRandom([1, 2, 3])); // => 2
```
